# WWW2025 repair utilities

These scripts reconstruct canonical item metadata when the raw-video/ASR id sequence drifts from the title/source/interaction sequence.

Run them through the documented workflow in [`docs/dataset_repair.md`](../../docs/dataset_repair.md). `VIDEO_ID_ALIGNMENT_NOTE.md` is executable input to `fix_items_asr_raw_offset.py`; do not simplify its piecewise mapping to one global offset.
